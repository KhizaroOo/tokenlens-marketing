# TokenLens — Marketing

Static marketing assets for the TokenLens product.

## Contents

| File | Description |
|---|---|
| `tokenlens-overview.html` | Self-contained HTML presentation deck (12 slides). No build step required — open directly in any browser. |

## Usage

```bash
# Just open in a browser — no server needed
start tokenlens-overview.html      # Windows
open tokenlens-overview.html       # macOS
```

## Presentation Controls

| Key | Action |
|---|---|
| `← →` or `Space` | Previous / next slide |
| `F` | Toggle fullscreen |
| Click right/left edge | Next / previous slide |
| Swipe | Touch navigation |

## Stack

Plain HTML + CSS + vanilla JS. Zero dependencies. Zero build step.
